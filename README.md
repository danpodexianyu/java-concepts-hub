# Java Concepts Hub

### 一、注释

**注释**不参与编译和运行，只是对代码进行解释和说明。

Java中的注释分为三种：

1. 单行注释
2. 多行注释
3. 文档注释

#### 1.单行注释

```java
// 单行注释
int foo; // 也可以写在代码后面
```

#### 2.多行注释

在代码中，有时候会使用大量的段落去解释说明某一段代码的逻辑，而这一段逻辑作为一个整体，我们可以直接使用多行注释。

```java
/* 多行注释 */
```

#### 3.文档注释

**文档注释**是针对于类、方法、成员变量等等解释说明的注释。

```java
/** 文档注释 */
```
